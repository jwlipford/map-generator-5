# map-generator-5
 Generates a randomized rectangle of characters representing a map with different characters representing water, lowlands, and hills

### Description
 The map is procedurally generated. In each *mutation*, a random line crosses the map, and the land on one side of the line increases in elevation. A few thousand mutations are recommended. The user sets two elevations that represent the boundaries between water and land and between lowland and hills.

### History
 As the final project for CSCI 330: Programming Language Structures, which I took in my last semester of college, each student took six weeks to learn the basics of a language of their choice that they had not used before and to make a presentation and small program showing their understanding of the language. I chose Rust. Many years earlier, when teaching myself C# in high school, I had written four similar map generator programs, so I thought it would be fun to write a fifth one for that class.
